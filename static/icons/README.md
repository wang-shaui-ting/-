# TabBar 图标存放说明

请将以下 6 个图标文件放入当前文件夹（`static/icons/`）：

## 所需图标（81×81 像素 PNG）

| 文件名               | 用途          | 颜色             |
| -------------------- | ------------- | ---------------- |
| `home.png`           | 首页 - 未选中 | 灰色 `#999999`   |
| `home-active.png`    | 首页 - 选中   | 夸克蓝 `#1677FF` |
| `discuss.png`        | 讨论 - 未选中 | 灰色 `#999999`   |
| `discuss-active.png` | 讨论 - 选中   | 夸克蓝 `#1677FF` |
| `profile.png`        | 个人 - 未选中 | 灰色 `#999999`   |
| `profile-active.png` | 个人 - 选中   | 夸克蓝 `#1677FF` |

## 图标风格建议

- **尺寸**：81×81 像素（微信 tabBar 推荐尺寸）
- **风格**：简约线条图标，与夸克 UI 风格一致
- **格式**：PNG（透明背景）
- **线条宽度**：约 3~4px

## 图标含义

- **首页**：🏠 房子 / 三角形屋顶 + 矩形主体
- **讨论**：💬 气泡对话框
- **个人**：👤 圆形头像 + 身体弧线

## 注意

图标文件已在 `app.json` 中配置：

```json
"iconPath": "static/icons/home.png",
"selectedIconPath": "static/icons/home-active.png"
```

请确保文件名**完全一致**（区分大小写），否则 TabBar 图标将无法显示。
